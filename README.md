# Crowd No Crowd — Offline OSM Demo

Проект демонстрирует работу офлайн-картографического приложения с анализом пользовательской активности и рекомендацией маршрута.

---

## Цель проекта

Показать:
- работу карт без интернета (offline-first подход)
- моделирование поведения пользователей
- применение машинного обучения (кластеризация)
- интеллектуальную рекомендацию маршрута

---

## Возможности

- Офлайн-карта
- Онлайн-режим (OSM)
- Генерация пользователей
- Анализ плотности (DBSCAN)
- Рекомендация движения
- Desktop-приложение

---

## Как работает система

1. Генерируются пользователи (сценарии)
2. DBSCAN находит зоны плотности
3. Выбирается целевая точка
4. Строится направление движения
5. Отображается карта и рекомендации

## ⚙️ Установка

```bash
git clone https://github.com/your-username/osm-offline-demo.git
cd osm-offline-demo
```

### Создание виртуального окружения

```
python -m venv .venv
```

### Активация окружения
#### Windows

```
.venv\Scripts\activate
```

#### Linux/Mac

```
source .venv/bin/activate
```
### Установка зависимостей
```
pip install -r requirements.txt
```
### Запуск
```
python -m app.main
```
